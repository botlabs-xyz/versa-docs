## Troubleshooting

### Slash commands are missing

- Confirm VERSA is installed in server integrations.
- Wait briefly after invite/update for Discord sync.
- Re-invite using the official invite link if needed.

### Moderation commands fail

- Check VERSA role placement in **Server Settings → Roles**.
- Confirm required moderation permissions are enabled.
- Check channel-specific overrides.

### Logs are not posting

- Confirm log channel is configured.
- Ensure VERSA can send messages/embed in that channel.
- Verify the channel still exists and is visible to bot role.

### Automation does not trigger

- Confirm module is enabled.
- Confirm triggers/conditions are configured.
- Test in a controlled channel first.

## What to send support

- server name
- command used
- expected result
