## First-Time Setup

### 1. Invite VERSA

Use the official invite:

[Invite VERSA](https://discord.com/api/oauth2/authorize?client_id=1200483261533650966&permissions=8&scope=applications.commands+bot)

### 2. Configure Role Placement

Open **Server Settings → Roles** and place VERSA high enough to perform moderation actions you enable.

### 3. Set Log Channel

Configure a log channel in a staff-visible text channel.

### 4. Enable Required Modules

Enable moderation and automation modules needed for your server policy.

### 5. Run Validation

- Test `/help`
- Test one moderation command in safe staff channel
- Confirm log entries are posted

## Setup Checklist

- Commands are visible in server
- Role and channel permissions are correct
- Logs are posting in configured destination
- Automation modules are enabled intentionally
