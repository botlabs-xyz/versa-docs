## Privacy Policy

VERSA processes Discord metadata required to provide moderation and automation features.

### Data used

- Discord server, channel, role, and user IDs
- Command usage metadata
- Moderation and automation event metadata

### Why data is used

- Execute moderation workflows
- Run configured automation modules
- Improve reliability and support diagnostics

### Sharing

Data is not sold. Processing may involve infrastructure required to operate VERSA.
